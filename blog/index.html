    线程传参

    创建 POSIX 线程需要使用函数 pthread_create()，例如：

    pthread_t threads[N];
for (int i = 0; i < N; i++) {
  int *datap = (int *) malloc(sizeof(int));
  *datap = i;
  pthread_create(&threads[i], NULL, print, datap);
}

    4 个参数依次为存储线程地址、属性、执行的函数名、向线程传递的参数的指针。

    上述代码问题在于内存分配了 N 次，但是没有及时释放，内存分配的指针被下一次覆盖而丢失，存在内存泄露问题。
