1 History

在 8086 之前是 8008 等处理器，8086 是一个 16 位处理器，具有 16 位的数据位，Intel 推出 8086 后由于当时 16 位的外设普遍较贵，又推出了 8 位兼容的 8088，8088 被 IBM PC 所采用并且操作系统是 MS-DOS，从此开启了 Wintel 的康庄大道。此后，又推出后继系列处理器 80186、80286、80386、80486、Pentium 等。

2 Architecture

2.1 Registers

由于系列最初的处理器 8086 是 16 位处理器，所以 AX、BX、CX、DX、SI、DI、IP、SP、BP 等寄存器都是 16 位的。之后推出 8088 为 8 位兼容版，所以又把 AX、BX 等划分高低位，如 AH、AL，允许 8 位的存取。后继系列处理器中，32 位处理器的寄存器代号加上前缀变为 EAX、EBX 等即表示 32 位拓展的寄存器，64 位则是 RAX、RBX 等。

以下是寄存器代号和用途。

通用寄存器（general-purposed register）：

label meaning
AX accumulator, store arithmetic results
BX base addressing register
CX counter in loop operations
DX point to data in IO operations

段寄存器（segment register）：

label meaning
CS code segment register
DS data segment register
ES extra data segment register
SS stack segment register
FS, GS additional data segment registers

索引寄存器（index register）：

label meaning
SI source index, for moving strings and array index
DI destination index

标志寄存器及标志位（flags register and flags）：

label meaning
EFLAGS flags register
ZF zero flag
OF overflow flag
CF carry flag
PF parity flag
SF sign flag
DF direction flag

其他：

label meaning
IP instruction pointer
SP stack pointer
BP base pointer

3 Segments

8086 处理器有 20 位地址，可以表示为 0x00000 ～ 0xFFFFF，一个段可表示范围为 0x0000 ～ 0xFFFF，因此一个段最大为 64 KB 即 216，段的起始地址低 4 位应为 0x0，例如 0x00000、0x00010、0x00020（可能是为了和偏移量的位数一样）。因此，段可以用段基址省略末尾 0 和偏移量表示，即 <base>:<offset> ，例如 1AB2:0091 表示 1AB20 至 1AB20 + 0091。

4 Piplining

8086 的流水线分两个阶段取指（fetch）和执行（execute）。

5 Instructions

5.1 DIV

被除数是 EDX:EAX 的合并，除以 DIV 指令给定的寄存器，商存在 EAX，余数存在 EDX。如：100 / 6 = 16 … 4，100 放在 EAX 里，EDX 清零，6 放在 ECX，除 ECX，EAX 为 16，EDX 为 4。 

mov edx, 0        ; clear dividend
mov eax, 0x8003   ; dividend
mov ecx, 0x100    ; divisor
div ecx           ; EAX = 0x80, EDX = 0x3

5.2 MUL

乘数存在 EAX，乘以 MUL 指令给定寄存器，结果存在 EDX:EAX 的合并里。如 100 * 10 = 1000，100 放在 EAX，10 放在 ECX，乘以 ECX，结果 1000 存在 EDX:EAX 的合并里。 

mov edx, 0
mov eax, 100
mov ecx, 10
mul ecx

5.3 CMP

比较指令，与条件跳转一起使用。

cmp dst, src ZF CF
dst = src 1 0
dst < src 0 1
dst > src 0 0

5.4 JE, JNE, JG, JGE, JB, JBE, JMP

条件跳转指令，与比较指令结合使用。 

cmp eax, 0
je counternowzero

5.5 LOOP

循环，默认 EDX 为计数器，每执行一次计数器减一，当计数器减至零不循环。 

mov ecx, 5
startloop:
  loop startloop
