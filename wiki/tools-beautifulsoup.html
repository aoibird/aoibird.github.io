只是单纯地摘录文档。

1 基本使用

from bs4 import BeautifulSoup

soup = BeautifulSoup(html)
soup = BeautifulSoup("<html>data</html>")
soup = BeautifulSoup(open("index.html"))

可以传入字符串也可以传入文件句柄。文档会被转化为Unicode。

  • soup.prettify() : 格式化
  • soup.title : 得到 <title> 标签
  • soup.title.name
  • soup.title.string
  • soup.title.parent.name
  • soup.p
  • soup.p['class'] : 得到第一个 <p> 标签的 class 属性
  • soup.a
  • soup.find_all('a') : 找到所有 <a> 标签，可迭代
  • soup.find(id = 'link3')
  • soup.get_text() : 得到文档的所有文字内容

2 安装

通过 apt 、 pip 或 easy_install 安装BeautifulSoup4。

可选解析器：

  1. Python标准库 : BeautifulSoup(markup, "html.parser")
  2. lxml HTML解析器 : BeautifulSoup(markup, "lxml")
  3. lxml XML解析器 : BeautifulSoup(markup, ["lxml", "xml"])
  4. html5lib : BeautifulSoup(markup, "html5lib")

3 对象种类

  • Tag（标签）
    • Name : tag.name 获取，比如 soup.title.name
    • Attibutes : tag['class'] 或 tag.attrs 获取， tag['id'] = 2 修改， del tag['id'] 删除
      • 多值属性
  • NavigableString（可历遍字符串）
  • BeautifulSoup
  • Comment（注释及特殊字符串）

4 历遍文档树

  • 子节点
    • 通过标签的名字，点取法，比如 soup.body.b ，只能取得第一个标签
    • tag.contents : 将子节点以列表形式输出
    • tag.children : 可迭代
    • tag.descendants : 生成器，对子孙节点（包含子节点的子节点）历遍
    • tag.string : 仅有一个子节点时，返回子节点，多个子节点返回None
    • tag.strings : 生成器，用来历遍NavigableString
    • tag.stripped_strings : 生成器，去除多余的空白内容
  • 父节点
    • tag.parent : 返回父节点（BeautifulSoup对象）
    • tag.parents : 生成器，可历遍所有父节点
  • 兄弟节点
    • tag.next_sibling 和 tag.previous_sibling : 返回BeautifulSoup对象
    • tag.next_siblings 和 tag.previous_siblings : 返回生成器
  • 回退和前进
    • tag.next_element 和 tag.previous_element : 下一个或上一个被解析的对象
    • tag.next_elements 和 tag.previous_elements : 返回生成器

5 搜索文档树

  • 过滤器
    • 字符串：比如 soup.find_all('b') ，为查找所有 <b> 标签
    • 正则表达式：比如 soup.find_all(re.compile("..."))
    • 列表：比如 soup.find_all(["a", "b"]) 为查找所有 <a> 和 <b> 标签
    • True：比如 soup.find_all(True) 查找到所有的标签，但是不会返回字符串节点
    • 方法：定义方法过滤
  • find_all() : find_all(name, attrs, recursive, text, **kwargs)
    • name 参数 : 按标签名字搜索，比如 soup.find_all("title") 为查找名为title的标签
    • keyword 参数 : 按标签的属性搜索，比如 soup.find_all(id = 'link2') 和 soup.find_all(class_ = 'link2')
    • text 参数 : 按字符串内容搜索，比如 soup.find_all(text = "Elsie") 和 soup.find_all(text=True)
    • limit 参数 : 限制搜索返回结果数量，比如 soup.find_all("a", limit=2)
    • recursive 参数 : 是否递归搜索，比如 recursive=False
    • 简写形式 : soup.find_all("a") -> soup("a")
  • 同理可知
    • find()
    • find_parent() 、 find_parents()
    • find_next_sibling() 、 find_next_siblings()
    • find_previous_sibling() 、 find_previous_siblings()
    • find_next() 、 find_all_next()
    • find_previous() 、 find_all_previous()
  • CSS选择器 : soup.select("head > title")

6 修改文档树

  • 对属性赋值
  • tag.append() : 比如 soup.a.append("Bar")
  • BeautifulSoup.new_string() 和 BeautifulSoup.new_tag()
  • tag.insert() : 比如 tag.insert(1, "but did not endorse ")
  • elem.insert_before() 和 elem.insert_after() （可以是标签或者文本节点）
  • tag.clear() : 清除标签的内容
  • elem.extract() : 从文档树移除元素并返回（可以是标签也可以是文本节点）
  • elem.decompose() : 完全销毁
  • elem.replace_with() : 替换
  • elem.wrap() : 对元素进行包装并返回结果，比如 soup.p.string.wrap(soup.new_tag("b"))
  • elem.unwrap() : 与之相反的操作

7 输出

  • 格式化输出 : soup.prettify()
  • 压缩输出 : str()=或=unicode()
  • 得到文本内容 : get_text() ，比如 soup.get_text("|", strip=True)
