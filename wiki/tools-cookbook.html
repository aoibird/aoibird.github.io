3 Verilog

  • Icarus Verilog also iverilog is a compiler for verilog.
  • GTKWave is a tool to show wave of signals.

3.1 Install

pacman -S iverilog gtkwave

3.2 Usage

Compile: 

iverilog my_module.v my_module_tb.v -o my_module

Run: 

./my_module

Create a wave: 

vvp my_module

Show the wave: 

gtkwave my_module.vcd

3.3 Testbench

module my_module_tb;
  initial begin
    $dumpfile("my_module.vcd");
    $dumpvars(0, var1, var2, var3 /*...*/);
  end

  reg clk, rst;
  reg var1, var2 /*...*/;
  my_module my_module1(var1, var2, var3 /*...*/);

  initial begin
    $monitor("At time %t, %0d %0d ...", $time, var1, var2 /*...*/);
    rst = 1;
    clk = 1;
    #100 rst = 0;  // assign rst then deassign
    forever #100 clk = ~clk;  // invert clk every 100T
  end

endmodule

6 Wget

6.1 Options

  • -S, --server-response : Print the headers and responses.
  • -N, --timestamping : Turn on time-stamping.
  • -m, --mirror : Turn on options suitable for mirroring. This option turns on recursion and time-stamping, sets infinite recursion depth and keeps FTP directory listings. It is currently equivalent to -r -N -l inf --no-remove-listing.
  • -K, --backup-converted : When converting a file, back up the original version.
  • -k, --convert-links : After the download is complete, convert the links in the document to make them suitable for local viewing.
  • -E, --adjust-extension : Append suffix.
  • -p, --page-requisites : Download all the files that are necessary to properly display a given HTML page.
  • -H, --span-hosts : Turn on spanning hosts.
  • -D DOMAIN-LIST, --domains=DOMAIN-LIST : Set domains to be followed. DOMAIN-LIST is a comma-separated list of domains. Note that it does not turn on -H. e.g. -Ddomain1.com,domain2.com.

6.2 Download an entire website

wget --random-wait -r -p -e robots=off -U mozilla http://www.example.com

Options:

  • -p parameter tells wget to include all files, including images.
  • -e robots=off you don't want wget to obey by the robots.txt file
  • -U mozilla as your browsers identity.
  • --random-wait to let wget chose a random number of seconds to wait, avoid get into black list.

Other options:

  • --limit-rate=20k limits the rate at which it downloads files.
  • -b continues wget after logging out.
  • -o $HOME/wget_log.txt logs the output

From: download an entire website - commandlinefu.com

11 Powerline

Installation (macOS and Homebrew): 

brew install python
pip3 install powerline-status

Zsh: 

powerline-daemon -q
POWERLINE_BASH_CONTINUATION=1
POWERLINE_BASH_SELECT=1
. /usr/local/lib/python3.7/site-packages/powerline/bindings/zsh/powerline.zsh

Tmux: 

source "/usr/local/lib/python3.7/site-packages/powerline/bindings/tmux/powerline.conf"

Fonts: 

git clone https://github.com/powerline/fonts
cd fonts
./install.sh
