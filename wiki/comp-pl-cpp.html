1 Object-oriented

主要概念：抽象、封装、重载、继承、多态。

面向对象的思想是通过定义类来实现对“对象”这一概念的抽象，类将数据和代码封装在一起构成对象的属性和方法。类可以通过继承派生出子类，也叫派生类，派生类的方法可以与基类不同，这是用多态实现的运行时选择调用函数。重载即对操作符重新定义，对不同的类执行特定操作。

1.1 Define

类有属性和方法，属性一般为 private ，方法一般为 public ，方法主体可以写在类的内部也可以写在外部，写在内部写法就是普通函数，写在外部方法名加上类名： 

class Fruit {
private:
  int weight;
public:
  void peel(void);
};

void Fruit::peel() {
  printf("peeling a base class fruit\n");
}

类有构造函数和析构函数，构造函数在对象创建时调用，做一些初始化的工作，析构函数则是销毁对象释放内存，析构函数可以显式调用，通常被会自动调用，两个特殊函数没有返回值，且要与类同名： 

Fruit::Fruit() {};
Fruit::~Fruit() {};

1.2 Inheritance

继承就是复用父类的属性和方法，继承的写法如下： 

class Fruit {};
class Apple : public Fruit {};

1.3 Operator overloading

可以重定义操作符对类的操作，比如两个水果对象相加是两个对象的重量相加： 

class Fruit {
public:
  int operator+(Fruit &f);
private:
  int weight;
};

// 可以使用：
// Fruit f1, f2, f3;
// f3 = f1 + f2;
int Fruit::operator+(Fruit &f) {
  return weight + f.weight;
}

1.4 Override

在 C++ 中需要显式覆盖添加 virtual 限定符： 

class Fruit {
public:
  void peel(void) { printf("peeling a base class fruit\n"); }
};

class Apple : public Fruit {
public:
  virtual void peel(void) { printf("peeling an apple\n"); }
};

2 Standard Library

C++ 标准库中已经有常见的数据结构可以直接使用。

2.1 Vector

类似于动态数组，长度可变的数组，常用操作有 push_back 、 erase 、 insert 、 begin 、 end 、 size ： 

#include <vector>
using namespace std;
vector<int> v;
v.push_back(1);
v.push_back(2);
v.erase(v.begin() + 1);
v.insert(v.end(), 3);

for (int i = 0; i < v.size(); i++);

2.2 Stack

常用操作 push 、 pop 、 top 、 empty 、 size ： 

#include <stack>
using namespace std;
stack<int> s;
s.push(1);
s.top();
s.pop();
s.empty();
s.size();

2.3 Queue & Priority Queue

同理，常用操作有 push 、 empty 、 front 、 pop 。

2.4 Structure

结构体可以像类一样在内部定义函数。
